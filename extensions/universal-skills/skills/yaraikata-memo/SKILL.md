# Yaraikata Memo

## Overview

Run the work with a helper in the background. Every 5 minutes, give a short update with expected time. Explain how it works in simple words. If something goes wrong, give simple next steps.

## Steps

1. Confirm the goal, what "done" means, and where results should appear.
2. Start a helper to do the work while you stay responsive.
3. Every 5 minutes, check the helper and report:
   - What is done
   - What is next
   - Expected time (rough)
4. Ask a second helper to explain how it works. Share the explanation in plain words.
5. If there is a problem:
   - Say what happened in one short sentence
   - Offer 1-2 simple choices to fix it
   - Ask which choice to try

## Output style

- Keep updates short and clear
- Avoid jargon
- Prefer simple words
- Use the same shape each time
