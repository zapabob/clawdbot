# 実装ログ - 2026-03-20

## 概要

- `qwen-hakua-core` カスタムモデルを Ollama に登録した。
- RTX 3060 GPU 上で全レイヤーが動作するよう `num_gpu 99` を設定した。

---

## 手順詳細

### 1. GGUF ファイル確認

```
C:\Users\downl\Downloads\Qwen3.5-9B-Uncensored-HauhauCS-Aggressive-Q8_0.gguf
```

存在確認済み（約9.5GB、Q8_0量子化）。

### 2. 初回モデル登録

```powershell
cd C:\Users\downl\Desktop\clawdbot-main3\clawdbot-main
ollama create qwen-hakua-core -f scripts/Modelfile.hakua
```

- GGUF 取り込み完了（sha256: `99e7f220...`）
- `qwen-hakua-core:latest` として登録（9.5 GB）

### 3. GPU オフロード設定追加

`scripts/Modelfile.hakua` に `PARAMETER num_gpu 99` を追記:

```
PARAMETER num_gpu 99   # 全レイヤーを GPU に展開
PARAMETER temperature 0.8
PARAMETER top_p 0.9
...
```

モデルを再作成（GGUF レイヤーは既存を再利用、パラメータレイヤーのみ差し替え）。

### 4. 環境情報

| 項目         | 値                             |
| ------------ | ------------------------------ |
| GPU          | NVIDIA GeForce RTX 3060        |
| VRAM         | 12,288 MiB (空き: ~10,635 MiB) |
| モデルサイズ | 9.5 GB (Q8_0)                  |
| num_gpu      | 99（全レイヤー GPU 展開）      |

VRAM に全レイヤーが収まるため CPU オフロードは発生しない見込み。
