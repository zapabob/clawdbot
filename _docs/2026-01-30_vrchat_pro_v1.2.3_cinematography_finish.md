# 2026-01-30 VRChat Pro v1.2.3 Cinematography Finish

## 概要

VRChat 2025.3.3 (Open Betaを含む) の最新仕様に完全に準拠した「撮影監督向け仕上げ」を行いました。公式の物理カメラパラメータを網羅し、ファイル探索ロジックを最適化することで、撮影現場での即応性と信頼性を確保しました。

## 修正内容

### 1. 完全なカメラ制御パラメータ (Full Camera Suite)

以下のパラメータを追加・補正し、公式スライダーと完全に一致させました。

| パラメータ        | 範囲            | 単位/備考                      |
| :---------------- | :-------------- | :----------------------------- |
| **Smoothing**     | **0.1 - 10.0**  | 手ブレ補正強度                 |
| **FocalDistance** | **0.0 - 10.0**  | 焦点距離 (以前の1.0上限を撤廃) |
| **LookAtMe**      | **0.0 - 1.0**   | ヘッドトラッキング追従率       |
| **GreenScreen**   | **1 - 9**       | 背景合成モード                 |
| **FlySpeed**      | **0.05 - 10.0** | ドローン移動速度               |
| **TurnSpeed**     | **0.05 - 10.0** | ドローン回転速度               |

これにより、オートフォーカス的な演出や、スムーズなドローン撮影の速度制御まで、エージェント経由で精密に行えるようになりました。

### 2. 探索ロジックの最適化 (Optimized Discovery)

アバター設定探索の Glob パターンを `*/Avatars/${avatarId}.json` (深度2) に限定しました。

- **Before**: `**/Avatars/...` (再帰探索で LocalLow 全体を舐める可能性があり、遅くて危険)
- **After**: `OSC/usr_xxx/Avatars/...` の階層構造のみをピンポイントで探索。
  この変更により、不要なファイル誤検知のリスクが消滅し、探索速度が向上しました。

### 3. プロ向け運用ヒント (UX Polish)

`/vrc_reload` 実行時に、成功メッセージに加えて「パラメータが反映されない場合は VRChat 側の Reset OSC Config を使うこと」というヒントを追加しました。これにより、トラブルシューティングの初動が早くなります。
