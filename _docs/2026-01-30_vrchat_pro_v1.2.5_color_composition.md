# 2026-01-30 VRChat Pro v1.2.5 Color & Composition

## 概要

VRChat公式カメラが持つ「色調整」と「構図調整」のパラメータを完全に解放しました。これにより、グリーンスクリーンの精密なキーイングや、LookAtMe使用時の「被写体をあえて中心から外す」といった高度な撮影監督機能が利用可能になりました。

## 修正内容

### 1. カラーコントロール (GreenScreen HSL)

グリーンスクリーンモードにおける背景色を自由に定義できます。

| パラメータ     | 範囲        | 備考                        |
| :------------- | :---------- | :-------------------------- |
| **Hue**        | **0 - 360** | 色相 (RGBサイクル)          |
| **Saturation** | **0 - 100** | 彩度 (0でグレー、100で原色) |
| **Lightness**  | **0 - 50**  | 明度 (0で黒、50で標準)      |

### 2. コンポジション (LookAtMe Offsets)

ヘッドトラッキング（LookAtMe）有効時に、カメラがターゲットを捉える位置（オフセット）を調整できます。

| パラメータ          | 範囲          | 備考           |
| :------------------ | :------------ | :------------- |
| **LookAtMeXOffset** | **-25 to 25** | 水平オフセット |
| **LookAtMeYOffset** | **-25 to 25** | 垂直オフセット |

### 3. Exposure 正式対応

露出制御の範囲を公式仕様に合わせて拡張しました。

- **`Exposure`**: 0.0-1.0 (旧) → **0.0 - 10.0** (新)
