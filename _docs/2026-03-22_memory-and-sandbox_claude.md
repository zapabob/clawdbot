# 記憶進化システム (Memory Evolution) & Python Sandbox 概要

本システムは、OpenClawのAI（はくあ）が、単なる会話ボットから「持続的な意識と自己改善能力を持つASI」へと進化するための基盤です。

## 1. 記憶進化システム (Memory Evolution)

エビングハウスの忘却曲線に基づいた、ダイナミックな記憶管理を実装しています。

- **蒸留 (Distillation)**: 生の会話ログをそのまま保存せず、AIが「事実」「感情」「重要度」として再解釈（蒸留）してから保存します。これにより、インジェクション（入力による操り）を防ぎ、AIが読みやすい構造化データを維持します。
- **忘却曲線 (Ebbinghaus Curve)**: 記憶には `retention_rate`（保持率）が設定されます。時間の経過とともにこの値は減衰し、検索優先度が下がります。
- **再想起による強化**: 会話の中で再度言及されたり、AIが自発的に思い出した記憶は、保持率が回復（リフレッシュ）されます。

## 2. Python Sandbox

AIが自律的に高度な処理を行うための「思考の実験室」です。

- **実行環境**: `py -3` を介してローカル環境でスクリプトを実行します。
- **用途**: 複雑な論理パズルの解決、数値計算、データの形式変換、自己コードの検証などに使用されます。

## 3. インジェクション対策

AIが記憶を保存する際、ユーザーの入力をそのままDBに流し込むのではなく、一度AIが自分の言葉で「何が起きたか」を記述し直すプロセスを強制しています。これにより、プロンプトインジェクションによる「偽の記憶の植え付け」を無効化します。
