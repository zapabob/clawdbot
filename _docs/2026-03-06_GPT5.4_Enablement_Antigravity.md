# 2026-03-06 GPT-5.4 Enablement & ACPX Fix Implementation Log - Antigravity

## 概要

OpenClaw 内部で `openai-codex/gpt-5.4` を完全に利用可能にするためのランタイム修正と設定更新を実施しました。

## 実施内容

### 1. ACPX プラグインのスキーマ修正

- `extensions/acpx/openclaw.plugin.json` に `codexHarness` プロパティを追加しました。これにより、`openclaw.json` でこの設定を有効にした際のバリデーションエラーを解消しました。

### 2. ランタイムソースコードのパッチ

ハードコードされていた `gpt-5.3-codex` 前提のロジックに `gpt-5.4` を追加しました。

- `src/agents/model-catalog.ts`: モデル ID 定数の追加。
- `src/agents/model-forward-compat.ts`: 前方互換性（既存モデルからのクローン）ロジックに 5.4 を追加。
- `src/agents/live-model-filter.ts`: モダンモデルリストに `gpt-5.4` を追加。
- `src/auto-reply/thinking.js`: `xhigh` 思考レベル（推論機能）を有効にするモデルリストに `openai-codex/gpt-5.4` を追加。

### 3. 設定更新とビルド

- `~/.openclaw/openclaw.json` を更新し、`primary` モデルに `openai-codex/gpt-5.4` を設定、タイムアウトを 600 秒に延長しました。
- `pnpm build` を実行し、ソースコードの変更を `dist` に反映させました。

### 4. 検証

- `node openclaw.mjs config validate` にて設定の正当性を確認。警告なし。
- `node openclaw.mjs models list` にて `openai-codex/gpt-5.4` が認識され、`default` および `fallback` に指定されていることを確認。

## 結論

`openai-codex/gpt-5.4` は現在 OpenClaw 内で正常に認識され、優先モデルとして動作する準備が整いました。
