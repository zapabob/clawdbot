# 2026-02-01 OS起動時自動実行の設定

## 概要

PCの電源投入（OS起動）時に、Clawdbotが自動的に起動するように設定しました。

## 設定内容

- **起動バッチファイル**: `start_clawdbot.bat` をプロジェクトルートに作成。
  - プロジェクトディレクトリへの移動 (`cd /d`)
  - `pnpm dev` によるボットの起動
- **スタートアップ登録**: Windowsのスタートアップフォルダ (`shell:startup`) に上記バッチファイルへのショートカットを作成。
  - ショートカット名: `Clawdbot.lnk`
  - 保存先: `%APPDATA%\Microsoft\Windows\Start Menu\Programs\Startup`

## 検証内容

- **ファイル存在確認**: `start_clawdbot.bat` の作成を確認。
- **ショートカット確認**: `Test-Path` により、スタートアップフォルダ内にショートカットが存在することを確認。

## 備考

- 次回のWindows起動時から、ボットが自動的にコマンドプロンプトで立ち上がります。
- 自動起動を停止したい場合は、スタートアップフォルダ内の `Clawdbot` ショートカットを削除してください。
