# 2026-02-05 公式リポジトリ同期 (Upstream Sync)

## 概要

公式リポジトリ `openclaw/openclaw` の最新機能、脆弱性更新、バグフィックスを取り込みつつ、このフォーク独自の機能を維持した状態でマージを完了しました。

## 実施内容

### 1. Upstream 設定

- 公式リポジトリを `upstream` リモートとして追加しました。
  ```bash
  git remote add upstream https://github.com/openclaw/openclaw
  ```

### 2. マージ処理

- 履歴が大幅に異なっていたため、`--allow-unrelated-histories` を使用してマージを開始しました。
- 8874件のコミットを取り込みました。

### 3. コンフリクト解消

計345ファイルのコンフリクトを Python スクリプトおよび手動で解消しました。

**解消方針:**

- **ドキュメント・設定ファイル**: 両方の変更を取り込む（Union）
- **独自機能**: LINE統合機能、Codex/OpenCode連携など、独自機能に関わる部分はすべて維持。
- **機能が同等な場合**: 独自機能側の実装（バッチスクリプト等）を優先。
- **新規ファイル**: 公式の新規ファイルはすべて受け入れ、独自の新規ファイルはすべて維持。

### 4. 主な更新

- 公式側の2025年最新機能（A2UI, Voice Wake, 14以上のチャンネルサポート等）のベースラインを更新。
- セキュリティアップデートの適用。

## 今後の対応

- 大規模なマージを行ったため、起動確認を推奨します。
- 必要に応じて、`npm install` または `pnpm install` を実行して依存関係を更新してください。
