# OpenCode CLI Skill

Use OpenCode CLI to perform software engineering tasks.

## Usage

```
/opencode <task>
```

Describe a software engineering task to complete.

```
/opencode analyze <file>
```

Analyze code in a file.

```
/opencode refactor <file> <instructions>
```

Refactor code based on instructions.

```
/opencode test <file>
```

Generate tests for a file.
